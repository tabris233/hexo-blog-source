# python import研究

​	`python`用了好久了, 对它的`import`规则一直都很懵,借此机会梳理下.


<!-- more -->


### 首先是导包.

导入同目录下的包, 或者安装好的第三方模块, 或者python自带的模块

```python
import module_name
```

导入当前目录中某一文件夹里面的`python`模块

```python
from dir import module_name
```
